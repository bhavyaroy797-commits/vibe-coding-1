# Trops Page

A high-performance, responsive landing page designed for the "trops" beauty brand. This project focuses on clean aesthetics and user experience, seamlessly transitioning content across different devices.

## 🚀 Features
* **Mobile-First Design:** Fully responsive layout ensuring a perfect look on desktops, tablets, and phones.
* **Minimalist UI:** High-focus typography and clean design inspired by modern beauty e-commerce trends.
* **Interactive Elements:** Smooth interface transitions powered by Vanilla JavaScript.

## 🛠 Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Flexbox/Grid for layout, custom styling, and responsive media queries.
* **JavaScript (ES6):** Logic for interactive components.
